라이브 온! 와카바

(Live on! Wakaba)

PC용 방치형 스트리머 육성 시뮬레이션 게임

컨셉: PC에서 키우는 나만의 미소녀 다마고치. 나무 정령 컨셉의 캐릭터 '이부키 와카바'를 SSS급 스트리머로 육성하고, 특별 일러스트를 모두 해금하자.

사용 기술: Python/Pygame

이미지: Stable Diffusion WebUI 생성

소개 페이지

아르카나 코드 v1.0

(Arcana Code v1.0)

AI가 해석해 주는 무료 타로카드 프로그램

컨셉: 진정한 무작위 카드 뽑기가 불가능하다는 LLM의 결점을 보완한 쪽집게 타로카드 소프트웨어. 어떤 질문이든 완벽 해결. 해석은 최신 AI가 알아서 해 준다!

사용 기술: 프론트엔드(HTML5, CSS3, JS ES6+), 백엔드(Node.js, Express.js), LLM API(Google Gemini 1.5 Flash), 이미지(Stable Diffusion)

에테르 튜너

(Aether Tuner v1.0)

웹페이지로 굴리는 나만의 라디오닉스 머신

컨셉: 나만의 커스텀 소원성취 주파수를 원한다면? 사용자가 입력한 의도를 기반으로 화이트/브라운 노이즈를 생성하는 스태틱 웹사이트 형식의 라디오닉스 머신. 매력적인 비주얼라이저와 깔끔한 UI로 언제 어디서나 사용 가능.

사용 기술: 프론트엔드(HTML5, CSS3, JS ES6+), 오디오(Web Audio API), 라이브러리(p5.js)
